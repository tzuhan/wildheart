# Purpose

[Wildheart](https://wildheart.cc) is a transparent platform that aggregates and ranks the fundraising needs of Taiwanese wildlife rescue and conservation organizations, helping donors identify where support is most urgently needed.

Feedback and suggestions are welcome via this [Google Form](https://docs.google.com/forms/d/e/1FAIpQLSdMLif96qT409dSQo_1ix3cR5pe7-1LxTEwHaV1u6GFAyO6zQ/viewform?usp=dialog_)

# 專案宗旨

[Wildheart](https://wildheart.cc) 是一個致力於資訊透明的平台，彙整並呈現台灣野生動物救援與保育組織的募款需求，透過排行方式協助捐款者了解哪些團隊目前最需要支持。

歡迎透過此 [Google 表單](https://docs.google.com/forms/d/e/1FAIpQLSdMLif96qT409dSQo_1ix3cR5pe7-1LxTEwHaV1u6GFAyO6zQ/viewform?usp=dialog) 提供意見與建議。
